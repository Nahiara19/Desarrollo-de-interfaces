Cómo jugar
Movimiento del personaje:

El personaje sigue la posición del dedo índice detectado por la cámara. Asegúrate de que tu mano esté visible en la cámara.
Recolectar monedas:

Mueve el personaje para interceptar las monedas que caen. Cada moneda recolectada aumenta tu puntuación.
Evitar obstáculos:

Si chocas con un obstáculo, pierdes una vida. Pierdes el juego al quedarte sin vidas.
Usar el escudo:

Di el comando de voz "usar" para activar un escudo que te protege de los obstáculos durante un choque.
Objetos especiales:

Recoge el objeto especial (si aparece) para ganar una vida extra.
Controles
Teclado:

Presiona q para salir del juego.
Comando de voz:

Di "usar" para activar el escudo.
Mecánicas del juego
Progresión:

Aumenta de nivel cada 5 puntos.
Los obstáculos y las monedas se mueven más rápido a medida que avanzas de nivel.
Vidas:

Comienzas con 3 vidas.
Cada colisión sin escudo te hace perder una vida.
Puedes ganar vidas recolectando el objeto especial.
Dificultad:

La frecuencia de aparición de obstáculos y monedas depende del nivel.
